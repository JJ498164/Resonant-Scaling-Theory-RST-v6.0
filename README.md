# Resonant-Scaling-Theory (RST) v6.2.1

## Abstract
RST is a spectral model of neural recovery. It posits that re-synchronization in disrupted networks is constrained by the **Spectral Gap ($\lambda_2$)**. This repository provides a formal Python framework to validate the **Square-Root Recovery Law** and the **Neural Reynolds Number**.

## Claim Hierarchy
* **Tier 1 (Strong):** Recovery time $\tau$ follows $\lambda_2^{-1/2}$ scaling.
* **Tier 2 (Emergent):** Frequency clustering is topology-dependent (e.g., $f^*$ clusters in Theta/Gamma bands).
* **Tier 3 (General):** Bottlenecks are emergent properties of metabolic throttling.

## Initial Validation Results
Initial sweeps across scale-free and small-world topologies (n=100 to 400) confirm a stable relationship between the Spectral Gap and recovery latency. 
- **Mean Frequency Detected:** 1.55 Hz (Theta Pacing)
